Примерный код на языке Python для модуля Ретопология (Retopology), который пересоздает геометрию объекта-меша с нуля:
Этот код выбирает объект-меш для ретопологии, создает новый объект-меш для ретопологии, устанавливает его в режим ретопологии, пересоздает геометрию нового объекта-меша с помощью добавления новых ребер и сохранения копии геометрии существующего объекта-меша в новый объект-меш. Конечно, этот пример может быть изменен и расширен в зависимости от конкретных требований проекта.
